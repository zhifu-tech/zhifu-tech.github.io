工具: mac 截屏指南

发表于 2025-10-13 分类于 工具
mac 截屏指南
📖 来源： 腾讯元宝 点击查看原文

当然！Mac 系统有非常强大且便捷的截屏功能。以下是所有常用的截屏快捷键，分为几个类别，方便你记忆。

一、核心快捷键（最常用）

这些快捷键会将截图保存在桌面（默认位置，可更改）。

快捷键 功能说明 截图预览（如有）
⇧ Shift + ⌘ Command + 3 全屏截图：截取整个屏幕。 全屏图像
⇧ Shift + ⌘ Command + 4 区域截图：光标变为十字线，拖选区域进行截图。 所选区域图像
⇧ Shift + ⌘ Command + 5 调出截屏工具栏（见第二部分详解） -

二、高级快捷键（区域截图的延伸）

在按下 ⇧ Shift + ⌘ Command + 4 后，还可以使用以下高级技巧：

操作 功能说明
按下 空格键 模式切换为窗口截图。移动光标到某个窗口上，窗口会高亮显示，点击即可单独截取该窗口（带漂亮阴影效果）。
按住 空格键（拖动） 锁定所选区域，然后整体移动选区位置。
按住 Shift 键 锁定选区宽度或高度，只允许单向移动。
按住 Option 键 以起点为中心，等比例缩放选区。

三、使用截屏工具栏 (⇧ Shift + ⌘ Command + 5)

这是最直观的方式，尤其适合录制屏幕。按下后会屏幕底部会出现一个控制条：

截屏选项（前三个图标）：

  1. 捕获整个屏幕：同 Command+Shift+3。
  2. 捕获所选窗口：点击后，再点击某个窗口即可截图。
  3. 捕获所选部分：同 Command+Shift+4，可自定义区域。

录屏选项（后两个图标）：
4. 录制整个屏幕
5. 录制所选部分

点击工具栏上的 “选项”，你可以进行重要设置，例如：

  • 保存到：更改保存位置（如桌面、文稿、剪贴板）。
  • 定时器：设置 5 秒或 10 秒后开始截屏/录屏。
  • 显示/隐藏鼠标指针等。

四、直接截图到剪贴板（不保存文件）

如果你希望截图后直接粘贴到微信、Word 等应用中，而不生成文件，可以在以上所有快捷键中额外增加一个 Control 键。

例如：

  • Control + ⇧ Shift + ⌘ Command + 3：全屏截图到剪贴板。
  • Control + ⇧ Shift + ⌘ Command + 4：区域截图到剪贴板（之后同样可以按空格键截取窗口）。

总结表格

目标 快捷键 备注
全屏截图（保存文件） ⇧ Shift + ⌘ Command + 3 最常用
区域/窗口截图（保存文件） ⇧ Shift + ⌘ Command + 4 按空格可截窗口
调出功能全面的工具栏 ⇧ Shift + ⌘ Command + 5 推荐新手，功能最全
全屏截图（到剪贴板） Control + ⇧ Shift + ⌘ Command + 3 直接粘贴使用
区域截图（到剪贴板） Control + ⇧ Shift + ⌘ Command + 4 直接粘贴使用

小贴士：在 macOS Mojave 或更高版本中，截图后屏幕右下角会短暂显示一个缩略图，点击它可以快速对截图进行标记、编辑或分享。
