Venus 日志模块

  • VENUS
  • LIBS
  • LOGGER

基于 kotlin-logging 的轻量级多平台日志框架。

特性

  • 🚀 多平台支持: 支持 Android、iOS、JVM 等平台
  • ⚡ 高性能: 使用懒加载，只有在启用相应日志级别时才会执行字符串拼接
  • 🎯 简单易用: 提供统一的 API，无需关心底层实现
  • 📱 平台适配: 自动适配不同平台的日志输出方式
  • 🏷️ 标签管理: 支持复合标签和标记功能

日志级别

  • trace: 最详细的日志，通常只在开发时使用
  • debug: 调试信息，用于开发过程中的问题诊断
  • info: 一般信息，记录程序运行的重要节点
  • warn: 警告信息，表示潜在的问题
  • error: 错误信息，表示程序执行中的错误

最佳实践

  1. 使用懒加载: 对于包含复杂计算的日志消息，使用 lambda 表达式
  2. 合理的标签命名: 使用有意义的标签名称，便于日志过滤和查找
  3. 适当的日志级别: 根据信息的重要性选择合适的日志级别
  4. 异常处理: 记录异常时，同时记录异常对象和描述信息
  5. 使用标记: 对于需要特殊分类的日志，使用标记功能进行分组
  6. 全局日志实例: 对于框架级别的日志，使用全局 logger 实例
